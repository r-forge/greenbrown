text.ObjFct {ModelDataComp}R Documentation

Convert objective functions to text

Description

Convert objective functions to text.

Usage

## S3 method for class 'ObjFct'
text(x, which = c("Cor", "RMSE", "MEF"), sep = ", ", digits = 3)

Arguments

x

an object of class ObjFct

which

which objective function metrics sshould be converted to text?

sep

separation between metrics

digits

digits for rounding

Details

No details.

Author(s)

Matthias Forkel <matthias.forkel@geo.tuwien.ac.at> [aut, cre]

References

No reference.

See Also

ObjFct

Examples


obs <- rnorm(100) # "observations"
sim <- obs + rnorm(100, 0, 0.8) # simulation
of <- ObjFct(sim, obs)
of

# convert objective functions to text:
text(of)
text(of, which="KGE")
text(of, which=c("R2", "MEF", "KGE"), sep=" ")
