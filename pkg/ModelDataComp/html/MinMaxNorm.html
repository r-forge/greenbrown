Normalize to a range

Description

Normalizes values to a given range.

Usage

MinMaxNorm(x, ext = c(0, 1), ...)

Arguments

x

values

ext

new minimum and maximum values

...

further arguments (unused)

Details

No details.

Value

vector with normalized values

No reference.

Examples


x <- rnorm(10, 0, 2)

plot(x, n)
