FitLogistic {ModelDataComp}R Documentation

Fit logistic functions

Description

Fit additive or multiplicative logistic functions to a response variable.

Usage

FitLogistic(x, y, additive = FALSE, method = c("genoud"), ntry = 50, 
    ...)

Arguments

x

predictor variables

y

response variables

additive

additive or multiplicative (default) fit?

method

Method to be used for optimization of fit parameters. "genoud" uses genetic optimization (see genoud). Can be also one of "Nelder-Mead", "BFGS", "CG", "L-BFGS-B", "SANN", "Brent" (see optim). If two methods are provided, the second method is applied on the best result of the first method.

ntry

number of inital trials for strating parameter sets in the optimization of fit parameters. A higher number of trials yields usually more robust results but takes longer.

...

further arguments for genoud or optim

Details

No details.

Value

An object of class 'FitLogistic' which is actually a list.

Examples


# 1D example
x <- 1:1000
y <- Logistic(c(1, 0.01, 500), x) + rnorm(1000, 0, 0.01)
plot(x, y)
fit <- FitLogistic(x, y)
lines(x, fit$predicted, col="red")
fit$par # fitted parameter

# performance of fit
of <- ObjFct(fit$predicted, y)
of
plot(of)
