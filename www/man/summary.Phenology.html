summary.Phenology {greenbrown}R Documentation

Print a summary of phenology metrics

Description

The function prints a summary the results from Phenology.

Usage

## S3 method for class 'Phenology'
summary(x)

Arguments

x

Object of class 'Phenology' as returned from function Phenology

Author(s)

Matthias Forkel <mforkel@bgc-jena.mpg.de>

Examples

# load a time series of NDVI (normalized difference vegetation index)
data(ndvi)
plot(ndvi)
	
# calculate phenology metrics
phen <- Phenology(ndvi)
phen
print(phen)

# plot phenology metrics
plot(phen)
