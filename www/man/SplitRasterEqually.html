SplitRasterEqually {greenbrown}R Documentation

Splits a raster in equal-area parts

Description

This function splits a raster object in parts with ~ equal area.

Usage

SplitRasterEqually(data.r, n)

Arguments

data.r

raster, raster brick or raster stack.

n

number of parts

Value

the function returns a list of raster layers

Author(s)

Matthias Forkel <mforkel@bgc-jena.mpg.de> [aut, cre]

Examples
