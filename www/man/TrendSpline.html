TrendSpline {greenbrown}R Documentation

Trend estimation based on a smoothing splines

Description

The function computes a non-linear trend based on smooth.spline.

Usage

TrendSpline(Yt, ...)

Arguments

Yt

univariate time series of class ts

...

additional arguments (currently not used)

Value

The function returns a list of class "Trend".

Author(s)

Matthias Forkel <mforkel@bgc-jena.mpg.de>

See Also

stl

Examples

# load a time series of NDVI (normalized difference vegetation index)
data(ndvi)
plot(ndvi)
	
# calculate trend on mean annual NDVI values
trd <- TrendSpline(ndvi)
trd
plot(trd)
